C++ 通讯录管理系统
“每天编程一小时，记录从零到一的 C++ 进阶之路。” 


🚀 本项目是基于 C++ 基础语法实现的通讯录管理系统，是我在学习 C++ 过程中的一个重要里程碑（12/100 挑战计划）。

📖 项目简介
这是一个简洁、高效的命令行通讯录管理系统，旨在练习和巩固 C++ 的核心基础知识。通过这个项目，我深入实践了以下技术点：

结构体 (struct)：定义联系人信息与通讯录结构。

数组与指针：管理联系人列表及其内存操作。

函数模块化：将各个功能封装为独立函数，提高代码复用性。

流程控制：利用 switch-case 和循环实现交互菜单。

✨ 主要功能
✅ 添加联系人：支持输入姓名、性别、年龄、电话及家庭住址。

✅ 显示联系人：一览所有已保存的联系人信息。

✅ 删除联系人：根据姓名精准查杀，动态维护列表。

✅ 查找联系人：通过姓名快速检索详细资料。

✅ 修改联系人：针对已有联系人进行信息更新。

✅ 清空通讯录：一键重置，清空所有数据。

✅ 退出系统：安全退出当前管理程序。

🛠️ 技术栈
语言：C++

工具：支持 g++ 编译器, Visual Studio, VS Code 等

平台：跨平台（Windows/Linux/macOS）
